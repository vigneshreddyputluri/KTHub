# KTHub

KTHub is a Knowledge Transfer Hub: a collaborative platform where learners, instructors, professionals, universities, and organizations can publish, learn from, improve, and reuse structured knowledge transfer modules.

The product idea is similar in spirit to GitHub, but instead of source code repositories, the main unit is a **KT**: a practical learning module with lessons, assignments, teaching notes, contributions, reviews, and adoption workflows.

## Current Delivery Stage

This repository currently contains the first SDLC baseline:

- Product vision and scope
- Initial software requirements
- MVP backlog
- Architecture proposal
- Static browser prototype

## Open The Prototype

Open this file in a browser:

```text
app/index.html
```

No build step is required for the current prototype.

## Primary User Modes

- **Learner mode**: discover KTs, study modules, complete practical assignments, track progress.
- **Instructor mode**: learn how to teach a topic, adapt lesson plans, curate assignments.
- **Creator mode**: publish KTs, accept contributions, manage versions.
- **Institution mode**: organize KTs for colleges, universities, companies, departments, and communities.
